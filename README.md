# LeetCode Solutions

## Description

This repository contains my personal solutions to **LeetCode** problems, with a primary focus on the **NeetCode 250** roadmap. As I continue solving problems on LeetCode, I plan to add additional problem sets to this repository.

The purpose of this repository is **not** to provide the most optimal or shortest possible solutions. Instead, it documents my learning journey and the approaches I developed while studying data structures and algorithms and preparing for technical interviews. As my understanding improves, I may revisit earlier solutions to optimize or refine them, while preserving the progression of my problem-solving skills.

## Languages

At the moment, all solutions are implemented in **Python**, and the main `NeetCode 250` directory contains the Python implementations.

If I solve the same problems in other programming languages in the future, they will be added as separate directories with descriptive names, such as:

* `NeetCode 250 - java`
* `NeetCode 250 - cpp`
* `NeetCode 250 - javascript`

## Purpose

This repository aims to:

* Track my progress through the NeetCode 250 roadmap.
* Practice writing clean, readable, and maintainable code.
* Strengthen my algorithmic thinking and problem-solving skills.
* Document my learning journey over time.

Feel free to explore the solutions, compare different approaches, or use them as a reference while studying. Keep in mind that these solutions reflect my understanding at the time I solved each problem, so they may not always represent the most optimal approach.
